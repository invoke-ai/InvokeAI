## الوظيفة الأساسية للملف (High-level Purpose)

هذا الملف يُعرّف **مسارات API لطابور الجلسات (Session Queue Router)**. إنه يُوفر واجهة REST API لإدارة طابور معالجة طلبات توليد الصور. يشمل العمليات الأساسية: إضافة دفعات (Enqueue)، وإلغاء، وحذف، وتفقد الحالة، وإعادة المحاولة. يمكن وصفه بأنه **لوحة التحكم للعمليات** التي تتحكم في تدفق طلبات التوليد.

## شرح المدخلات والمخرجات (Inputs/Outputs)

### المدخلات:
- **queue_id:** معرّف الطابور
- **batch:** الدفعة المراد إضافتها
- **item_ids:** قائمة معرّفات العناصر
- **batch_ids:** قائمة معرّفات الدفعات
- **destination:** وجهة العناصر

### المخرجات:
- **SessionQueueItem:** عنصر في الطابور
- **EnqueueBatchResult:** نتيجة الإضافة
- **SessionQueueAndProcessorStatus:** حالة الطابور والمعالج
- **BatchStatus:** حالة الدفعة

## تفصيل الدوال والأساليب البرمجية (Functions & Methods)

### الدوال المساعدة:

#### `sanitize_queue_item_for_user()` - تصفية العناصر
```
دالة أمان لإخفاء الحساسية للمستخدمين غير المديرين.
```
- **المديرون ومالكو العناصر:** يرون كل شيء
- **المستخدمون العاديون:** يرون فقط الحالة والتوقيت

### مسارات API:

#### 1. `POST /{queue_id}/enqueue_batch` - إضافة دفعة
```
إضافة دفعة جديدة من طلبات التوليد إلى الطابور.
```
- **المعلمات:** `queue_id`, `batch`, `prepend`
- **النتيجة:** `EnqueueBatchResult` (201)

#### 2. `GET /{queue_id}/list_all` - قائمة جميع العناصر
```
الحصول على جميع عناصر الطابور.
```
- **المعلمات:** `queue_id`, `destination` (اختياري)
- **النتيجة:** `list[SessionQueueItem]`

#### 3. `GET /{queue_id}/item_ids` - معرّفات العناصر
```
الحصول على معرّفات العناصر فقط (خفيف).
```
- **المعلمات:** `queue_id`, `order_dir`
- **النتيجة:** `ItemIdsResult`

#### 4. `POST /{queue_id}/items_by_ids` - جلب بالمعرّفات
```
الحصول على عناصر بعدة معرّفات.
```
- **المعلمات:** `queue_id`, `item_ids`
- **النتيجة:** `list[SessionQueueItem]`

#### 5. `PUT /{queue_id}/processor/resume` - استئناف المعالج
```
استئناف معالج الجلسات (Admin only).
```

#### 6. `PUT /{queue_id}/processor/pause` - إيقاف المعالج
```
إيقاف معالج الجلسات مؤقتاً (Admin only).
```

#### 7. `PUT /{queue_id}/cancel_all_except_current` - إلغاء الكل
```
إلغاء جميع العناصر ما عدا الجارية حالياً.
```
- **المستخدمون العاديون:** يلغون عناصرهم فقط

#### 8. `PUT /{queue_id}/delete_all_except_current` - حذف الكل
```
حذف جميع العناصر ما عدا الجارية حالياً.
```

#### 9. `PUT /{queue_id}/cancel_by_batch_ids` - إلغاء بالدفعة
```
إلغاء جميع عناصر دفعات معينة.
```

#### 10. `PUT /{queue_id}/cancel_by_destination` - إلغاء بالوجهة
```
إلغاء جميع عناصر وجهة معينة.
```

#### 11. `PUT /{queue_id}/retry_items_by_id` - إعادة المحاولة
```
إعادة محاولة عناصر فاشلة.
```

#### 12. `PUT /{queue_id}/clear` - مسح الطابور
```
مسح جميع العناصر.
```

#### 13. `PUT /{queue_id}/prune` - تقليم الطابور
```
حذف العناصر المكتملة أو الفاشلة فقط.
```

#### 14. `GET /{queue_id}/current` - العنصر الحالي
```
الحصول على العنصر الجاري معالجته.
```

#### 15. `GET /{queue_id}/next` - العنصر التالي
```
الحصول على العنصر التالي دون تنفيذ.
```

#### 16. `GET /{queue_id}/status` - حالة الطابور
```
الحصول على حالة الطابور والمعالج.
```

#### 17. `GET /{queue_id}/b/{batch_id}/status` - حالة الدفعة
```
الحصول على حالة دفعة محددة.
```

#### 18. `GET /{queue_id}/i/{item_id}` - عنصر محدد
```
الحصول على عنصر محدد بالمعرّف.
```

#### 19. `DELETE /{queue_id}/i/{item_id}` - حذف عنصر
```
حذف عنصر محدد.
```

#### 20. `PUT /{queue_id}/i/{item_id}/cancel` - إلغاء عنصر
```
إلغاء عنصر محدد.
```

#### 21. `GET /{queue_id}/counts_by_destination` - العدّ بالوجهة
```
الحصول على عدد العناصر لكل وجهة.
```

#### 22. `DELETE /{queue_id}/d/{destination}` - حذف بالوجهة
```
حذف جميع عناصر وجهة محددة.
```
