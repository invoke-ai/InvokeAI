---

## الوظيفة الأساسية للملف (High-level Purpose)

هذا الملف هو **النقطة المركزية لإنشاء وتكوين تطبيق FastAPI** الذي يمثل الخادم (Backend Server) الرئيسي لمشروع InvokeAI. إنه يُعرّف التطبيق، ويُسجّل جميع مسارات API (Routers)، ويُضيف طبقات الوسيطات (Middleware)، ويعالج دورة حياة التطبيق من التشغيل إلى الإغلاق. باختصار، هذا الملف هو **الهيكل العظمي للخادم** الذي تتصل به واجهة المستخدم وجميع العملاء الخارجيين.

---

## المكتبات والحزم المستخدمة (Dependencies)

| المكتبة | الغرض |
|---|---|
| `fastapi` | إطار عمل بناء واجهات برمجة التطبيقات السريعة |
| `fastapi.middleware.cors` | معالجة سياسات CORS (الموصلية عبر الموارد) |
| `fastapi.middleware.gzip` | ضغط الاستجابات باستخدام GZip |
| `fastapi_events` | معالجة أحداث API باستخدام middleware خاص |
| `starlette` | الإطار الأساسي الذي يبنى عليه FastAPI |
| `uvicorn` | خادم ASGI لتشغيل التطبيق |
| `pathlib` | التعامل مع مسارات الملفات |
| `logging` | تسجيل الأحداث |
| `asyncio` | إدارة الحلقة غير المتزامنة (Event Loop) |

## شرح المدخلات والمخرجات (Inputs/Outputs)

### المدخلات:
- **ملف الإعدادات:** `app_config` المستخرج من `get_config()` (ملف `invokeai.yaml`)
- **طلبات HTTP:** جميع الطلبات الواردة من واجهة المستخدم أو العملاء (GET, POST, PUT, DELETE)
- **أحداث WebSocket:** عبر Socket.IO للتحديثات اللحظية

### المخرجات:
- **استجابة JSON:** بيانات API المطلوبة
- **استجابة HTML:** صفحات Swagger UI و Redoc للتوثيق
- **ملفات ثابتة:** واجهة المستخدم (React) من مجلد `dist`
- **WebSocket Events:** تحديثات حالة التوليد اللحظية

## تفصيل الدوال والأساليب البرمجية (Functions & Methods)

### 1. `lifespan(app: FastAPI)` - دالة دورة حياة التطبيق
```
ملاحظة: هذه دالة context manager غير متزامنة تُدار دورة حياة التطبيق.
```
- **التشغيل:** تستدعي `ApiDependencies.initialize()` لتهيئة جميع الخدمات والاتصالات
- **الإغلاق:** تستدعي `ApiDependencies.shutdown()` لإغلاق الخدمات بسلاسة

### 2. `SlidingWindowTokenMiddleware` - وسيط تحديث JWT
```
ملاحظة: فئة وسيط تمتد من BaseHTTPMiddleware.
```
- **الوظيفة:** تحديث رمز JWT (JSON Web Token) تلقائياً في كل استجابة مصادق عليها
- **الآلية:** تحقق من صحة الرمز، وأنشئ رمزاً جديداً مع تاريخ انتهاء جديد
- **الأولوية:** تعمل فقط على الطلبات الت mutating (POST, PUT, PATCH, DELETE)

### 3. `RedirectRootWithQueryStringMiddleware` - وسيط إعادة التوجيه
```
ملاحظة: فئة وسيط لتوجيه الطلبات.
```
- **الوظيفة:** إعادة توجيه الطلبات إلى الجذر مع query string إلى الجذر بدونها
- **السبب:** حل مشكلة توافق مع تطبيقات Gradio القديمة

### 4. `overridden_swagger()` - صفحة Swagger UI
- **المسار:** `GET /docs`
- **الوظيفة:** إرجاع صفحة Swagger UI المخصصة لتوثيق API

### 5. `overridden_redoc()` - صفحة Redoc
- **المسار:** `GET /redoc`
- **الوظيفة:** إرجاع صفحة Redoc البديلة لتوثيق API

### 6. تسجيل مسارات API (Routers)
```
ملاحظة: يتم تسجيل 16 مسار API رئيسي.
```
| المسار | الوصف |
|---|---|
| `/api/v1/auth` | المصادقة والتفويض |
| `/api/v1/utilities` | الأدوات المساعدة |
| `/api/v1/models` | إدارة النماذج |
| `/api/v1/download` | طابور التنزيل |
| `/api/v1/images` | إدارة الصور |
| `/api/v1/boards` | إدارة الألباب |
| `/api/v1/queue` | طابور الجلسات |
| `/api/v1/workflows` | سير العمل |
| `/api/v1/style_presets` | إعدادات الأنماط |
| `/api/v1/client-state` | حالة العميل |
| `/api/v1/custom-nodes` | العقد المخصصة |

### 7. تكوين الوسيطات (Middleware Stack)
- **CDS Middleware:** معالجة أحداث API
- **Socket.IO:** WebSocket للتحديثات اللحظية
- **CORS:** السماح بالموصلية عبر الموارد
- **GZip:** ضغط الاستجابات الأكبر من 1000 بايت

### 8. تحميل ملفات واجهة المستخدم
- **الملفات الثابتة:** `NoCacheStaticFiles` من `dist/` (واجهة React المجمّعة)
- **ملفات التوثيق:** من `static/docs/`
